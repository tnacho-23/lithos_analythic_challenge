Lithos Analytics

Prueba Técnica: CV Engineer Senior

Diseña, evalúa y despliega un Pipeline de Visión Artificial de alto rendimiento para el análisis granulométrico de rocas en entornos industriales críticos. Precisión, optimización y MLOps estricto son el estándar.

🪨

🗂️ Datos de Entrada (ZIP Adjunto)

En el archivo comprimido entregado encontrarás las imágenes necesarias para desarrollar y validar tus algoritmos. Estas simulan el entorno operativo real de las tolvas y cintas transportadoras.

📁

/Train

Set de imágenes para la exploración, desarrollo, ajuste de heurísticas o entrenamiento de modelos de segmentación.

📁

/valid

Set de validación para medir la generalización de tus métodos antes del empaquetado final en Docker.

Atención: La evaluación final se ejecutará con tu contenedor Docker apuntando a un directorio de testing completamente oculto. Tu solución no debe sobreajustarse a estos sets.

Arquitectura del Desafío

La prueba se estructura en tres módulos de complejidad progresiva. El enfoque metodológico es tan importante como el código final.

MOD 1Benchmarking de Métodos

Investigación aplicada y toma de decisiones.

🔬

Implementa y evalúa al menos tres (3) métodos distintos para segmentar rocas y calcular granulometría (ej. heurísticas clásicas, DL entrenado desde cero, modelos fundacionales).

Entregable Requerido:

Un reporte (Markdown o PDF) justificando métricas, hiperparámetros y analizando el trade-off (precisión vs costo computacional) de los métodos.

Automatización de métricas industriales.

Ponderación de la Evaluación

Evaluamos rigurosamente cada pilar del desarrollo Senior. Fallar en la contenerización o entregar código espagueti impactará fuertemente tu calificación.

🛡️ Reglas del Juego & Consejos de Optimización

  • ✔ Uso de IA y Datos Públicos: Totalmente permitido (Copilot, ChatGPT, Datasets externos para Fine-Tuning). Evaluamos tu integración y arquitectura, no tu memoria de sintaxis.
  • ✔ Optimización Senior: Se otorgará alta valoración al uso de herramientas como TensorRT, ONNX Runtime o estrategias de cuantización para Edge.
  • ✔ Código Limpio: Aplica tipado estático (Typing en Python), docstrings, manejo de excepciones y principios SOLID.
  • ✘ Cero Dashboards: No pierdas tiempo creando interfaces web (Flask/Streamlit) para los resultados. Centra tu esfuerzo en el backend computacional y la calidad del código.
